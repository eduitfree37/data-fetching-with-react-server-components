# Data fetching with React Server Components

For over a decade, React's core APIs have demonstrated remarkable stability since their initial release. The original component boundary, which facilitates state and lifecycle methods, continues to function effectively. The subsequent addition of Hooks ushered in a new phase of composition, empowering functional components with state and effects, and thus marking the second phase of React's evolution. Today, as Server Components are introduced, we embark upon React's third era of innovation.
